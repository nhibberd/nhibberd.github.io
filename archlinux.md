##### base System
`pacstrap -i /mnt base base-devel`

##### fstab
`genfstab -U -p /mnt >> /mnt/etc/fstab`
To use labels in place of uuid's replace `-U` with `-L`

##### chroot and configure
`arch-chroot /mnt /bin/bash`
